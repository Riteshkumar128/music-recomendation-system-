# music-recomendation-system-
Music Recommendation System built using Machine Learning and Streamlit. project contains three sub-projects with increasing complexity levels. It recommends songs using TF-IDF and cosine similarity, allows feature-based filtering using music attributes, and supports mood-based recommendations like Happy, Sad,  through an interactive web interface.


**Music Recommendation System using Machine Learning and Streamlit**

This repository contains a Music Recommendation System developed using Machine Learning techniques and deployed through a Streamlit web application. The project is designed to demonstrate different levels of complexity in building a recommendation system.

The repository includes **three sub-projects**, each built on the same concept of music recommendation but with increasing levels of programming and machine learning complexity:

**Level 1 – Basic Recommendation System**
A simple content-based recommendation system that suggests songs based on similarity between track name, artist name, and genre using TF-IDF vectorization and cosine similarity.

**Level 2 – Feature-Based Recommendation System**
An improved model that allows users to filter recommendations based on various musical attributes such as danceability, loudness, acousticness, energy, and other dataset features. This provides more personalized recommendations.

**Level 3 – Advanced Recommendation System with Mood Detection**
A more advanced implementation that introduces mood-based recommendations (e.g., Happy, Sad, Sleepy). The system analyzes audio features such as energy, sadness, and acousticness to suggest songs that match the user's mood.

All models are integrated into a **Streamlit-based interactive web interface**, allowing users to easily explore recommendations through song input, feature selection, and mood-based filtering.

**Technologies Used**

* Python
* Machine Learning
* TF-IDF Vectorization
* Cosine Similarity
* Pandas
* Scikit-learn
* Streamlit

This project demonstrates the gradual development of a recommendation system from a basic model to a more advanced and interactive machine learning application.
